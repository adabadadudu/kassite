## Features
- CrossPlatform(Linux,Windows,MacOS,iOS,...)
- Object-oriented
- Support functional programming
- Lightweight
- Fast & Powerful
- Easy to embed

# Build From Source
Prequistes :
- A C++ Compiler(should support `c++17` standard,gcc 8.1 is recommended)

for build :
```
make
```

for install(Linux):
```
make
make install
```
